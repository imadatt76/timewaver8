How to Choose

セッションの選び方

「どれを選べばいいかわからない」という方は、こちらを参考にしてみてください。

TYPE

01

目標や願望がはっきりしている

やりたいことや叶えたいことが明確な方は、波動調整コースがおすすめです。願望に直接コミットして、エネルギーを集中して整えます。

TYPE

02

悩みが多く、何から始めればいいかわからない

「何をしたいのかよくわからない」「迷っている」という方は、1年後のメッセージが向いています。タイムウェーバーが今のあなたに必要なことを引き出してくれます。

TYPE

03

気軽に、楽しみながら試してみたい

難しく考えずにエンタメ感覚で体験したい方には、おみくじ系セッションがぴったりです。「運」や「意識」の現在地を読み解きます。

TYPE

04

人間関係や先祖・ルーツとの関係が気になる

「家族や親との関係がずっと引っかかっている」「先祖のことが気になる」という方は、先祖エネルギー最適化コースへ。受け継いだエネルギーを整えます。

TYPE

05

心身・環境ごと、自分をトータルで整えたい

内側（思考・感情・潜在意識）も外側（身体・空間・環境）も、まとめて整えたい方に。内側と外側を整えるセッションや自分自身をパワースポットにするセッションが向いています。

TYPE

06

なんとなく重い・スッキリしない・引っかかりがある

原因はよくわからないけど、ずっとモヤモヤしている感じがある方は、自分を取り巻くエネルギーを整えるアプローチが合っているかもしれません。カルマの最適化・生霊と憑依の浄化のコースをご覧ください。

一番惹かれるものを、素直に選んでみてください。

タイプで選べなくてもかまいません。なぜかわからないけど気になる、最初に目が止まった、内容が頭から離れない——そういうフィーリングは、あなたの内側が「今ここに注目して」と伝えているサインかもしれません。
理由がなくても、直感で選んだセッションから始めてみるのもひとつの方法です。
